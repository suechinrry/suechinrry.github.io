数组

数组是存放在连续内存空间上的相同类型数据的集合。

  • 下标从0开始
  • 内存空间的地址是连续的
  • 数组的元素不能删除只能覆盖

二分法

使用前提：有序数组，无重复元素

left，right，middle只根据条件来改变left或right的值，middle的值自然会改变

写二分法，区间的定义一般为两种，左闭右闭即[left, right]，或者左闭右开即[left, right)。

  • 两种写法的区别while循环条件不同，r的赋值不同

  • 写法1：[left, right]

    class Solution {
    public int search(int[] nums, int target) {
        int l=0;
        int r=nums.length-1;
        int mid=0;
        while(l<=r){
            mid= l + ((r - l) / 2);
            if(target>nums[mid]){
                l=mid+1;
            }
            else if(target<nums[mid]){
                r=mid-1;
            }
            else{
                return mid;
            }            
        }
        return -1;
    }
}

    将mid=（l+r）/2换成mid= l + ((r - l) / 2)的原因:当l和r都很大时，会导致溢出。

class Solution {
    public int search(int[] nums, int target) {
        int l=0;
        int r=nums.length;
        int mid=0;
        while(l<r){
            mid=l+(r-l)/2;
            if(target>nums[mid]){
                l=mid+1;
            }
            else if(target<nums[mid]){
                r=mid;
            }
            else{
                return mid;
            }
        }
        return -1;
    }
}

class Solution {
    public int[] searchRange(int[] nums, int target) {
        int[] answer=new int[]{-1,-1};
        int l=0;
        int r=nums.length-1;
        int mid=0;
        int leftBorder=-2;
        int rightBorder=-2;
        //查找左边界
        while(l<=r){
            mid=l+(r-l)/2;
            if(nums[mid]>=target){
            r=mid-1;
            leftBorder=r;
            }
            else{
            l=mid+1;
            }
        }
        //查找有边界
        l=0;
        r=nums.length-1;
        mid=0;
        while(l<=r){
            mid=l+(r-l)/2;
            if(nums[mid]<=target){
            l=mid+1;
            rightBorder=l;
            }
            else{
            r=mid-1;
            }
        }        
        // 情况一
        if (leftBorder == -2 || rightBorder == -2) return new int[]{-1, -1};
        // 情况三
        if (rightBorder - leftBorder > 1) return new int[]{leftBorder + 1, rightBorder - 1};
        // 情况二
        return new int[]{-1, -1};
    }
}
